# Fashion MNIST Classification (Vibe Coding)

### 📌 Project Overview
This project implements deep learning models to classify the Fashion MNIST dataset, which contains 70,000 grayscale images across 10 clothing categories. The goal was to compare the performance of standard Neural Networks (ANN) against Convolutional Neural Networks (CNN).

### 🛠️ Tech Stack
* **Framework:** TensorFlow & Keras
* **Language:** Python
* **Libraries:** NumPy, Matplotlib (for visualization)
* **Platform:** Google Colab

### 📊 Key Steps
* **Data Preparation:** Normalized 28x28 pixel images for faster training.
* **ANN Architecture:** Built a fully connected dense network for baseline classification.
* **CNN Architecture:** Implemented Conv2D and MaxPooling layers to capture spatial features in the clothing images.
* **Visualization:** Plotted training loss and accuracy to monitor model "vibe" and performance.
